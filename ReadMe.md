How to get an Api key:

1. Go to https://aistudio.google.com

2. Login with your google account.
   
3. Click "Get API Key" on the left column of the website.

4. On the left click on Projects and make a new project.
   
5. On the left click on API Keys and Create an API Key.



How to insert the API Key:

Method 1 |

1. Open the app.

2. Click on Ai then Gemini Ai Api Key and copy and paste your key into the provided space.

Method 2 |

1. In your bookworm folder go into the _internal folder.

2. Find API.txt and copy and paste your API key into it.
